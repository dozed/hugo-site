The Scalatra Community

January 01, 0001

Quite a few big organizations have incorporated Scalatra as part of their production systems. You can see the ones we know about on our Built with Scalatra page.

Twitter

You can stay up to date on releases, new developments, and project news by following @scalatra on Twitter

Follow @scalatra

Github

Development takes place through our GitHub project. Come here for issue tracking, code reviews, and patches.

Google group

The scalatra-user Google Group is the preferred channel for Scalatra discussion so future users can learn from the archive. Support requests, suggestions, project announcements, and all other commentary related to Scalatra are welcome here.
